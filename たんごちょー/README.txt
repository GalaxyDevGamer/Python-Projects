説明書：

最初に言っておくが、リリース前のテスト版つまりこれな、これは説明書もソフトも全てタメ口になっている。

タメ口を直す方法は2つ
・金払って普通バージョン製作を依頼する。

・自分でpyファイルを開いて、コードを解読して正しく改造する。(余計なとこ消したとたんに動かなくなるから、構造を把握できるまで
お勧めしない)構造を把握するという意味でこっちを”推奨”する。

一から作ろうと思ったが、いざ構造を考えると、テキストファイル並みのスペースを求めるなら、講義マネージャーとあまり変わらん
のでベースは講義マネージャーにしてある。だから講義マネージャーの操作が分かれば大体操作できるだろう。
