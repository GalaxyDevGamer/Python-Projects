[このソフトについて]

・その名の通り、OSをイメージして開発しておりますが、まだまだOSには程遠いです。
	そういうことで、OSっぽく作ってあると思ってください。

【機能】
・現在の機能は家計などの「計算」に対する機能になります。
	また、これからも随時機能追加を行っていきます。

・このソフトでは、一人ひとりがそれぞれのデータを持てるようにアカウント作成形式を取ります。
	アカウント作成とは言っても、ソフトで色々な操作を行うためのものであって、
	個人情報を入力させたり、会員登録をさせたりする形式ではありません。

・アカウントファイルはソフトと同じフォルダに作成されるので、無闇にファイルをいじると、
	ソフトの操作に影響が出る場合がありますので全体に止めてください。

・β版と完全版には、恐らく機能にかなりの差がでると思います。
